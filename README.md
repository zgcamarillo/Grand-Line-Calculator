# Grand Line Calculator 
This is a One piece themed responsive and interactive calculator that handles errors involving the theme. This app was created using HTML, CSS, and JS.

## Features
-Addition, subtraction, multiplication and division <br>
-Use decimals, negative numbers, and percentage conversion <br>
-Graceful error handling <br>
-C (clear) button to change display to its original state <br>

 ## Errors 
 When errors happen, the calculator will display theme-related phrases. When the calulcator says "Luffy forgot a number!" it means the user forgot a number. When the calculator says "Sea King Attack!" it means the user put in an operation that is not valid.
